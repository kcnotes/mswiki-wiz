## Default Permission

Allows initializing the plugin and reading wz files

- `allow-init`
- `allow-get-png`
- `allow-get-json`
- `allow-parse-node`
- `allow-unparse-node`
- `allow-get-node-info`
- `allow-get-childs-info`
